# Doctor Appointment Booking System

## Introduction
The Doctor Appointment Booking System is designed to provide a digital solution for appointment scheduling. Patients can search for doctors, select available consultation slots, and receive notifications. Doctors can manage their schedules, while administrators can monitor appointment records and hospital activities.

## Technologies Used
- Frontend: React.js, HTML, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT, bcrypt

## Features
- User Authentication (Patient, Doctor, Admin)
- Doctor Search by Specialization
- Appointment Booking and Constraints
- Notifications System
- Role-based Dashboards (Admin, Doctor, Patient)

## Learning Outcomes
Targeting Full Stack Development, REST API Design, Role-Based Access Control, MongoDB schemas, and more.
